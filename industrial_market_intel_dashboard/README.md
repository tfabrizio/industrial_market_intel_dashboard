# Industrial Market Intelligence Dashboard

This project identifies and prioritizes high-value market opportunities across U.S. industries using structured economic data, clustering, and LLM-assisted recommendations.

## Features
- Opportunity scoring based on market size, growth, and concentration
- Market segmentation via KMeans clustering
- Executive-level recommendations using LLM outputs
- Interactive dashboard built with Streamlit
