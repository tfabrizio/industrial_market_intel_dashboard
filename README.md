# Industrial Market Intelligence Dashboard

**Live Link to Interactive Dashboard** : https://industrialmarketinteldashboard-mygy8flzts2pmhzbwvuqqc.streamlit.app/

## Overview
An interactive dashboard that identifies high-value industrial market opportunities using statistical scoring, clustering, and LLM-generated strategic recommendations.

Users can:
- Explore markets by state and industry
- Visualize opportunity positioning
- Interact with LLM-driven insights
